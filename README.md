# sprblog

## Описание

Реализация простого блога с помощью Spring MVC

Реализована функциональность:
- Вывода списка постов с пагинацией и фильтрацией по тегу
- Вывод конкретного поста со списком его комментариев
- Возможность редактировать содержимое поста, удалять и добавлять новые посты
- Возможность добавлять, редактировать и удалять комментарии
- Возможность ставить постам лайки

Вместо конфигурационного файла `web.xml` для настройки сервлетов используется класс `AppInitializer`
(см. [документацию](https://docs.spring.io/spring-framework/reference/web/webmvc/mvc-servlet/container-config.html))

Для автоматизации деплоя в сервлет контейнер Tomcat11 используется плагин `cargo-maven3-plugin`.
Он добавляет команду `mvn cargo:run` (см. ниже), которая автоматически скачивает Tomcat, размещает в нем собранный
war-ник и запускает его

## Команды для сборки, тестирования и запуска

- Тесты
```shell
mvn clean test
```

- Пересобрать
```shell
mvn clean package
```

- Запустить
```shell
mvn cargo:run
```

Главная страница со списком постов будет доступна по адресу `http://127.0.0.1:8080/sprblog/posts`
